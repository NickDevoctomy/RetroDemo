## RetroDemo

This is a rough little project containing some proof of concept code for creating a retro 8bit style UI using MonoGame.

### Controls

* Button
  - Utilising 9-slice scaling for flexible sizing
  - Supports hover and click states
  - Events for click handling
* Linear Retro Gradient
  - 2 colour support 
  - Customisable gradient stop count for more authentic 8bit look.
  - Customisable direction (horizontal/vertical/angled)
* Radial Retro Gradient
  - 2 colour support 
  - Customisable gradient stop count for more authentic 8bit look.
  - Customisable center point and radius
* Parallax Scroller
  - Customisable layers with individual speed settings
